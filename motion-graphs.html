Motion Graphs

Speed-Time and Distance-Time Lab

Motion graphs show how speed or distance changes over time. In a speed-time graph, the area under the graph gives distance. In a distance-time graph, the slope shows speed.

Speed-time graph: distance = area under the graph.
Average speed average speed = change in distance change in time
Distance from speed-time graph distance = area under the speed-time graph
Acceleration acceleration = change in speed change in time

Teacher Note

Each segment is piecewise. For speed-time, acceleration is constant inside each interval, so speed changes in a straight line. For distance-time, each segment uses a constant speed so distance changes linearly.

Start with a preset, or build your own journey using intervals like 0-10, 10-20, and 20-35.
